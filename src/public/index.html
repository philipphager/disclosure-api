Disclosure API

Welcome to the API documentation of Disclosure!

This is our Disclosure API. You can use this API to request, edit and remove libraries that are used in the Disclosure Android App. The Disclosure Android App searches for these libraries in installed Apps on the mobile device and analyzes what permissions they are using. The analyzed permissions are locally saved on the device, where the analysis was executed.

Libraries

List libraries

Lists all libraries.

GET Parameters Description Example
updatedSince (optional) Returns only libraries that were edited after the given date. Date format is ISO-8601. /libraries?updatedSince="2017-02-01T00:16:36.010Z"
page Responses are paginated. Navigate through all pages of the response by incrementing the page index. The first page has index 1. /libraries?page=2&limit=30
limit Number of returned libraries per page. /libraries?page=2&limit=30
GET
/libraries

Example request

$ curl https://morning-ridge-94362.herokuapp.com/libraries?updatedAt="2017-02-01T00:16:36.010Z"&page=2&limit=10

Create a library

Creates a new library.

Property Description
packageName the package name or id of the library
sourceDir the decompiled folder structure of the library
title the name of the library that is shown to the user
subtitle (optional) a one-line description of the library that is shown to the user
description (optional) description of the library that is shown to the user
websiteUrl link to the website of the library developer
type must be one of [ 'ADVERTISEMENT', 'ANALYTICS', 'SOCIAL', 'DEVELOPER' ]
POST
/libraries

Example request

curl --request PUT https://morning-ridge-94362.herokuapp.com/libraries \
  -d @data.json

Example request body

{
  "packageName": "com.facebook.stetho",
  "sourceDir": "com/facebook/stetho",
  "title": "Facebook Stetho",
  "subtitle": "",
  "description": "",
  "websiteUrl": "http://www.facebook.github.io/stetho",
  "type": "DEVELOPER"
}

  "updatedAt": "2017-02-23T14:33:23.370Z",
